Shell 基本命令

发表于 2017-09-13

注: 本资料系学习整理。

(1) shell版本与使用

  • 一般有6个虚拟控制台，用 到 切换，或 (n=2~6) 快速切换.
  1. bash,即 GNU 的 Bourne Again Shell，是 ubuntu 上常见的shell， 查看版本号
  • 有时候可以进行自动补齐.

  1. “；” 隔开不同的命令.

  2. “|” 管道，先进先出单向操作，将前一个命令的输出传到下一个命令作为输入，例如下面的命令查看硬件信息，并通过grep检索其中出现的字段

  • “<” 输入, “>”, “>>” 输出，下例将当前目录下所有文件名输出到result

(2) 文件管理

  • “~” 引用自己的主目录(/home/aby,其中 aby 是自己的账号名)
  • 常用通配符

*

任意字符

?任一字符

[a-f]a 到 f 中任一字符

\转义字符

  • cat 在命令行预览文件

下例预览文件并显示行号

  • more 分屏显示

从test.h中寻找include字符串并清空屏幕在顶端显示

  • less 按页显示，功能比前两者更强大

(3) 文件操作

  • cp 复制

将所有c语言源程序拷贝到Fi目录下

对目录进行操作(递归)

  • rm 删除

删除空目录

删除前进行提示

删除无法打开的文件，不进行提示

  • mv 移动或重命名

  • mkdir 创建目录

  • rmdir 删除空目录

  • pwd 显示当前路径

  • cd 更改工作路径, “..” 表示上层, “.” 表示当前层

    (4) 查找与检索

  • ls 查看目录

显示 “.” 开头的隐藏文件并列出目录下的所有子文件，用长格式

  • find 查找

需要指定查找路径，比如

  • locate 定位文件

可以直接查找文件名中出现的字符串

  • grep 文件内容检索

忽略大小写

  • ln 链接

创建软链接

  • touch 修改、创建文件

将 file2 的时间记录改为与 file1 一样

创建/修改文件，并指定时间

  • diff 比较两个文件内容(以及 diff3)

  • sort 对某个文件排序

    (5) 压缩文件与其他

  • tar 备份

打包、报告信息并指定备份文件

释放、报告信息并指定备份文件

使用 gzip 压缩或解压

通过bzip2压缩或解压

  • zip 压缩

压缩一个目录(-r)并解压

  • echo 显示文字

  • cal 显示日历

  • date 日期设置

  • clear 清屏(伪)

    (6) 奇怪的命令

  • ping 网络命令

向本机网络接口发送10个数据包

  • xrandr 强制改变屏幕分辨率

  • mount 挂载(未试验过！！！)

将C盘挂载到 /mnt 目录下并卸载

  • fdisk 查看硬盘

  • apt-get 下载软件

  • free 内存查看

用兆(M)作为显示单位

  • df 查看硬盘

  • ls+命令 查看各种接口

如 lsmod, lspci, dmesg, lsusb 等

  • chmod 更改访问权限

三个数字分别表示拥有者(u)、所属组(g)和其他人(o)(a表示这三者)的权限，可读(r=4)，可写(w=2)， 可执行(x=1)，可以使用 +，- 或 = 更改权限

对 .sh 文件一般需要赋予可执行权限

  • expr 计算表达式

  • let 计算表达式
